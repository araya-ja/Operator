Comparison Operators

  1. What is the result of print(5 == 3)

    • A) True
    • B) False

    Answer: B) False

  2. What is the result of print(7 != 2)

    • B) False

    Answer: A) True

  3. What is the result of print(10 > 5)

    Answer: A) True

  5. What is the result of print(9 >= 9)

    • A) True

    Answer: A) True

  6. What is the result of print(6 <= 3)

    • A) True

    Answer: B) False

  7. What is the result of print(2 == 2.0)

    • A) True

    Answer: A) True

    • A) True

    Answer: B) False

  9. What is the result of print(3 > 3)

    • A) True
    • B) False
