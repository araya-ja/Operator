Python Operators Exercises

Logical Operators

  1. What is the result of print(True and False)

    • A) True
    • B) False

    Answer: B) False

  2. What is the result of print(True or False)

    • A) True
    • B) False

    Answer: A) True

  3. What is the result of print(not True)

    • A) True
    • B) False

    Answer: B) False

  4. What is the result of print(False or not False)

    • A) True
    • B) False

    Answer: A) True

  5. What is the result of print((True and False) or True)

    • A) True
    • B) False

    Answer: A) True

  6. What is the result of print(True and not (False or True))

    • A) True
    • B) False

    Answer: B) False

  7. What is the result of print(False or (True and True))

    • A) True
    • B) False

    Answer: A) True

  8. What is the result of print(not (True and False))

    • A) True
    • B) False

    Answer: A) True

  9. What is the result of print(True or (False and True))

    • A) True
    • B) False

    Answer: A) True

  10. What is the result of print((not False) and (not True))

    • A) True
    • B) False

    Answer: B) False
