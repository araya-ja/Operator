Python Operators Exercises

Membership Operators

  1. What is the result of
    a = [1, 2, 3]
    print(2 in a)

    • A) True
    • B) False

    Answer: A) True

  2. What is the result of
    a = "hello"
    print('e' in a)

    • A) True
    • B) False

    Answer: A) True

  3. What is the result of
    a = [1, 2, 3]
    print(4 in a)

    • A) True
    • B) False

    Answer: B) False

  4. What is the result of
    a = (10, 20, 30)
    print(10 not in a)

    • A) True
    • B) False

    Answer: B) False

  5. What is the result of
    a = {'x': 1, 'y': 2}
    print('x' in a)

    • A) True
    • B) False

    Answer: A) True

  6. What is the result of
    a = {'x': 1, 'y': 2}
    print(1 in a)

    • A) True
    • B) False

    Answer: B) False

  7. What is the result of
    a = [1, 2, [3, 4]]
    print([3, 4] in a)

    • A) True
    • B) False

    Answer: A) True

  8. What is the result of
    a = "membership"
    print('ship' in a)

    • A) True
    • B) False

    Answer: A) True

  9. What is the result of
    a = [1, 2, 3]
    b = (1, 2, 3)
    print(2 in a and 3 in b)

    • A) True
    • B) False

    Answer: A) True

  10. What is the result of
    a = {'a', 'b', 'c'}
    print('d' not in a)

    • A) True
    • B) False

    Answer: A) True
